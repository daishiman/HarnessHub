規約・ポリシー

本サービスの利用規約とプライバシーポリシーです。

{{ t.l }}
最終更新日: 2026年7月1日
第1条（適用）

本規約は、Harness Studio（以下「本サービス」）の提供条件および利用者と運営者の権利義務関係を定めるものです。利用者は本規約に同意のうえ本サービスを利用するものとします。

第2条（アカウント）

利用者は、登録情報を正確に保つ責任を負います。アカウントは所属する組織（テナント）に紐づき、他社のデータにはアクセスできません。認証情報の管理は利用者の責任とします。

第3条（ハーネスの生成・利用）

本サービスが Claude Code を通じて生成するハーネスおよびヒアリングシートは、利用者が入力した情報に基づきます。生成物の業務適合性の最終確認は利用者が行うものとします。表示される削減効果は一般的な試算であり、成果を保証するものではありません。

第4条（禁止事項）

法令違反、第三者の権利侵害、リバースエンジニアリング、他テナントのデータへの不正アクセスを禁止します。

第5条（免責）

運営者は、本サービスの利用により生じた損害について、故意または重過失がある場合を除き責任を負いません。
最終更新日: 2026年7月1日
1. 取得する情報

氏名・メールアドレス・所属部署・単価などのアカウント情報、ヒアリング内容、ハーネスの実行ログ（回数・削減時間の集計に利用）を取得します。

2. 利用目的

本サービスの提供、ハーネスの生成・改善、利用状況とコスト削減効果の可視化、サポート対応のために利用します。

3. テナント分離

データは組織（テナント）単位で論理的に分離して管理され、各利用者は自社のデータのみ閲覧できます。

4. 第三者提供

法令に基づく場合を除き、本人の同意なく個人情報を第三者に提供しません。

5. お問い合わせ

個人情報の開示・訂正・削除のご請求は、privacy@harness-studio.example までご連絡ください。
